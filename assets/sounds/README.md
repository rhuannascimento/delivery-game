Coloque aqui os arquivos de som usados pelo jogo.

Para o som de colisão do jogador, adicione um arquivo chamado `bat_hit.ogg`.
Depois de adicionar, abra a cena `src/modules/player/Player.tscn` no Godot ou deixe a propriedade `crash_sound` no nó `Player` apontando para o arquivo.

Formato recomendado: OGG Vorbis (`.ogg`) 16-48 kHz.
